# Portfolio Website with Tailwind CSS

This project is the final result of a workshop on slicing Figma designs to HTML using Tailwind CSS.

## About

The design is inspired by a Figma community design template found [here](https://www.figma.com/design/lMrBIIogWesr1Jq81G7ZmR/Portfolio-UI---Web---Mobile--Community-?node-id=6-52&t=F8gmrbDbd790lCwC-0).

## Workshop Details

- **Location**: Tegal, Indonesia
- **Date**: November 4, 2025
- **Topic**: Figma to HTML conversion using Tailwind CSS

## Technologies Used

- HTML5
- Tailwind CSS
- Figma (Design Reference)

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Explore the responsive portfolio design
